# HORIZON projects (XML)

This is a mirror of the [HORIZON projects XML dataset](https://data.europa.eu/data/datasets/cordis-eu-research-projects-under-horizon-europe-2021-2027). Updates are pulled every night, creating a new commit in this repository if data has changed. No modifications are made except for pretty-printing of the XML files, which make it easier to see which parts of the data has changed.
